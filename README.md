📌 Project Overview

The Sports Management System is a Java-based GUI application designed to help students easily request sports equipment. The system allows students to browse and select available equipment, while the admin manages all requests and inventory.

This project is built using Java with GUI implementation and follows Object-Oriented Programming (OOP) principles.

✨ Features
👨‍🎓 Student Panel
View available sports equipment
Request equipment for use
Track request status (approved/rejected)
🛠️ Admin Panel
Add new sports equipment
Update existing equipment details
Delete equipment
Approve or reject student requests
View complete request history
💻 Technologies Used
Java
Java Swing / AWT (GUI)
Object-Oriented Programming (OOP)
🖥️ Project Functionality
Separate login access for Admin and Students
Interactive GUI interface for smooth user experience
Real-time request handling system
Inventory management system for equipment tracking
📁 How to Run the Project
Open the project in Visual Studio Code or any Java IDE
Compile all .java files
Run the main file (e.g., Main.java)
Use the GUI interface to interact with the system
